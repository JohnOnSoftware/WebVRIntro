选择 WebVR 开发框架

Three.js

为什么要用 ThreeJS

  • 已有很多开发案例
  • 无需学习很多底层技术 ( JS 技术)
  • 直接使用 WebGL 会很难调试
  • 很大，并且很清楚的 open source 库

Step 1: 创建一个场景 


// Create 3D objects.
var geometry = new THREE.BoxGeometry(0.5, 0.5, 0.5);
var material = new THREE.MeshNormalMaterial();
cube = new THREE.Mesh(geometry, material);

// Position cube mesh to be right in front of you.
cube.position.set(0, controls.userHeight, -1);

// Add cube mesh to your three.js scene
scene.add(cube);

Step 2: 添加光照 


var light = new THREE.AmbientLight( 0x404040 ); 
scene.add( light ); 
var light2 = new THREE.PointLight( 0xffffff, 1, 100 ); 
light2.intensity = 1; 
scene.add( light2 ); 
var light3 = new THREE.DirectionalLight( 0xffffff ); 
light3.position.set( 0, 1, 1 ).normalize(); 
scene.add(light3);

Step 3: 添加材质纹理 


// Tip: Use a spherical camera or your phone to capture a spherical photo
// Add an equirectangular panorama
var pano;
var loader = new THREE.TextureLoader();
loader.load('Greeley_pan_small_stars.jpg', onTextureLoaded);
function onTextureLoaded(texture) {
  var geometry = new THREE.SphereGeometry(1000, 32, 32);
  var material = new THREE.MeshBasicMaterial({
    map: texture,
    side: THREE.BackSide
  });
  pano = new THREE.Mesh(geometry, material);
  pano.position.y = 190;
  scene.add(pano);
}

Step 4: 添加动画 



var lastRender = 0
function animate (timestamp) {
  var delta = Math.min(timestamp - lastRender, 500)
  lastRender = timestamp
  // Apply rotation
  fit.rotation.y += delta * 0.0003
  fit.rotation.x += delta * 0.00005
  // Update VR headset position and apply to camera.
  controls.update()
  // Render the scene through the manager
  manager.render(scene, camera, timestamp)
  requestAnimationFrame(animate)
}

React VR: Facebook 出品

React VR > React Runtime > OVRUI > Three.js > WebVR & Browser

创建 ReactVR 应用的基本步骤

  1. Install Node.js, NPM
  2. Compatible VR Web Browser
  3. npm install -g react-vr-cli
  4. react-vr init MyReactVRSample
  5. npm start
  6. Open your browser at http://localhost:8081/vr
